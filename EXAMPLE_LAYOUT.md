### **Features:**
- **Status Bar:** Real-time battery monitoring and WiFi signal strength.
- **Dynamic Task List:** Automatically sorts active tasks first and marks completed ones.
- **Deadline Tracking:** Shows date and time for each task if provided.
- **Instant Web UI Access:** The QR code points to the device's local IP, allowing you to manage tasks from your smartphone without typing an address.
- **Dark Mode Support:** All elements invert for high-contrast viewing at night.
